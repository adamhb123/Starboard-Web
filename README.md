# Starboard Web
A web leaderboard for [@Mstrodl](https://github.com/Mstrodl/)'s [Slack Bot](https://github.com/Mstrodl/starboard)

## Getting Started
### Installation
Clone the repo, cd into the directory

    git clone sshblabalbablba && cd ./starboard-web
Install dependencies

    npm install
### Execution

    node bin/www
